Device Comparator

This is a comparator which helps you to comparing the spec between CharaChorder devices, Master Forge and Svalboard. The source code can be accessed here: https://github.com/andy23512/device-comparator.

Notices

  1. You could learn more differences between CC2 and CC1 from the CC2 Product Page.
  2. It seems to me that other unmentioned specs of CC devices would be similar to those of CC1, and they would be marked with an * in the comparison table.
  3. CCU and CC2.1 uses the same PCB as CC2, so some of the specs of them would be same to those of CC2.
  4. You could further confirm the specs on their Discord server if you have any questions.
  5. "???" are the specs which I am not sure about or cannot find information about. If you know of any helpful resources, please message me on Discord (Tangent Chang, @andy23512).
  6. If you prefer a comparison from my experience, check out my review articles.

    • Review to M4G
